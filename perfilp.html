Perfil de programador

Mis conocimientos en HTML me han permitido dominar diferentes etiquetas, que describo en la siguiente tabla. Estas habilidades pueden ser de gran valor para la empresa SpotiGT, si consideran contratarme.

Etiquetas que aprendí

Etiqueta Función
h1 a h6 Define encabezados de nivel 1 a 6, desde el más importante (h1) hasta el menos importante (h6).
p Define un párrafo de texto.
a Crea un enlace a otra página web o recurso.
br Inserta un salto de línea.
ol Define una lista ordenada.
ul Define una lista desordenada.
li Define un elemento de lista dentro de una lista ordenada o desordenada.
img Inserta una imagen en la página.
video Inserta un video en la página.
iframe Inserta un contenido de otra página web dentro de la página actual.
div Define una división o sección en la página.
table Define una tabla.
tr Define una fila dentro de una tabla.
th Define una celda de encabezado dentro de una tabla.
td Define una celda de datos dentro de una tabla.
